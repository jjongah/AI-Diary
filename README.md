# 나무의 서재

AI 서재지기와 대화하거나 직접 작성해 하루의 작은 장면을 남기는 로컬 저장 웹 일기입니다.

## 실행

```bash
npm install
npm run dev
```

브라우저에서 `http://127.0.0.1:5173`을 엽니다. 기록은 이 주소의 IndexedDB에 저장되므로 개발 중에도 주소와 포트를 바꾸지 않는 것을 권장합니다.

## Gemini 연결

프로젝트 루트에 `.env.local` 파일을 만들고 `.env.example`을 참고해 Google AI Studio API 키를 입력합니다.

```env
GEMINI_API_KEY=your_key_here
GEMINI_MODEL=gemini-3.5-flash
```

키가 없을 때는 화면과 전체 흐름을 시험할 수 있는 데모 응답이 사용됩니다. API 키는 브라우저 코드에 포함되지 않고 서버 라우트에서만 읽습니다.

## 데이터 보관

- 일기, 대화, 설정, 첨부 파일은 현재 브라우저의 IndexedDB에 저장됩니다.
- 브라우저 종료와 컴퓨터 재부팅 후에도 같은 브라우저와 같은 주소에서는 유지됩니다.
- 사이트 데이터 삭제, 브라우저 프로필 초기화 또는 기기 고장에 대비해 설정 화면에서 ZIP 백업을 정기적으로 만들어야 합니다.
- 복원은 기존 데이터를 병합하지 않고 전체 교체합니다. 복원 전에 현재 데이터를 자동으로 백업하도록 안내합니다.
